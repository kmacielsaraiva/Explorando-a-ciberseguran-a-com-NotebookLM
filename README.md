# Explorando-a-ciberseguran-a-com-NotebookLM
Este repositório contém um guia de estudos estruturado sobre Cibersegurança, desenvolvido com o auxílio do NotebookLM. O objetivo principal é estabelecer uma base teórica robusta, compreendendo os pilares da segurança da informação e criando um roteiro de aprendizado para guiar meus estudos futuros na área.
